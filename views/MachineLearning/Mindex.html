Machine Learning: Overview

In 1959, Arthus Samuel defined machine learning as the field of study that gives computers the ability to learn without being explicitly programmed. But it seems an informal definition of machine learning.

Tom Mitchell provided more modified form of definition of Machine Learning:

"A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E."

-Tom Mitchell

Machine Learning is a subset of artifical intelligence. In which computers analyse data, find hidden data patterns or relationships, and try to extract information to crack relations and provide insights. For humans, it is very hard to wrte the complicated programmes or algorithms but through machine learning, computers or gadgets learn and improve from experience without being explicit programmed. It has varios applications such as Image Processing, Data Mining, Healthcare, Video Games, Robotics, Finance and Text Analysis.
