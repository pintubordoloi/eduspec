Machine Learning

Machine Learning: Overview

In 1959, Arthus Samuel defined machine learning as the field of study that gives computers the ability to learn without being explicitly programmed. But it seems an informal definition of machine learning.

Tom Mitchell provided more modified form of definition of Machine Learning:

"A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E."

-Tom Mitchell

Machine Learning is a subset of artifical intelligence. In which computers analyse data, find hidden data patterns or relationships, and try to extract information to crack relations and provide insights. For humans, it is very hard to wrte the complicated programmes or algorithms but through machine learning, computers or gadgets learn and improve from experience without being explicit programmed. It has varios applications such as Image Processing, Data Mining, Healthcare, Video Games, Robotics, Finance and Text Analysis.

Types of Machine Learning



Machine learning can be categorized into three parts on the basis of its nature of learning, which are as follows-

1. Supervised Learning

Supervised Learning provides power tool to classify and process data using machine language. In supervised learning, using labelled data (dataset has features or labels), supervised machine learning algorithms predict the classification of other unlabelled data. It is the easiest paradigm of machine learning to understand and simplest to implement

It can be classified into two subcategories:


a. Regression

Regression is a supervised learning technique which deals with continous quantitative dataset, it is used in predicting, forecasting, and finding relationships between two quantitative features in which a few are independent of any factor and others are dependent on independent variables. It can be further classified into linear, polynomial and multiple linear, Support Vector Regression, Bayesian, Dicision-tree Regression, Random Forest Regression, Ridge Regression, Lasso Regression and Elastic Net Regression. It is used in weather forecasting, population growth prediction, market forecasting, life expectancy growth prediction, market forecasting, life expectancy estimation and real state price prediction.

b. Classification

classification involves grouping the data into various classes on the bsis of different features. It has different types of models like Logistic Regression, Random Forest, Naive Bayes, K-Nearest Neighbours, Support Vector Machines(SVM) and Decision Tree Classification.

2.Unsupervised Learning


Unsupervised Learning is the second subcategory of machine learning. In which machines deals with unlabelled and unclassified datasets which are used to train the algorithm. In unsupervised learning, the algorithms explore, try to identify some hidden pattern or structures within the data. The role of machine is grouping the unsorted data on the basis of similarties, differences and patterns without any previous training.

Types of Unsupervised Learning:

a. Clustering:

It is the task of dividing unlabelled entites into a number of various groups without any pre-trained model, these groups consist similar data points and these gorups used to make clusters.
There are many clustering algorithms in machine learning, which varies with its applications. Some of them are K-Means clustering, Hierarchical Clustering and Fuzzy C-Means Clustering.

Dimensionality Reduction:

It is the transformation of data to useful feature containing data from highly-featured data. In layman term, it is a technique that reduce the irrelevant data features. Some of the techniques are Principal Component Analysis, Linear Discriminant Analysis and Kernel Principal Component Analysis.

3. Reinforcement Learning

Reinforcement learning is a subcategory of machine learning in which software take actions in an environment in order to maximize the notion of cumulative rewards and punishments. In layman language, it involves models to learn from its past mistakes and then the machine tries to correct it in the next iteration. It is used in Robotics, Business, Gaming, Navigation, Data processing, creating training systems and aviation. Reinforcement learning has three main components

  1. Agent
  2. Environment
  3. Action

There a lot of algorithms within Reinforcement learning, which are applied in different domains, some of them are Q-Leaning, Deep Q Network, Policy Gradient and Markov Decision Process.

4.Semi-Supervised Learning

Semi-Supervised Learning is the fourth type of machine learning, which deals with both types of raw data, labelled and unlabelled data. In layman language, it is the hybrid of Supervised and Unsupervised learning which consist small amount of labelled data and significant amount of unlabelled data respectively. It is very useful with compare to Supervised learning because the process of labelling datasets is a time consuming process and expensive, while on the extra hand it consist the possibilities of emotional biasing too. It can use each type of method such as classification, regression and prediction.
